一、简介
该工程是我在2025-5月份完成，在FreeRTOS基础上，移植了+使用了XCMD命令行，完成串口交互，书写了部分链表代码，使用arudino编译2.3.4，xshell串口连接交互。
该链接为XCMD的gitee链接，它可以移植到其他单片机：https://gitee.com/two_salted_eggs/xcmd#%E4%BD%BF%E7%94%A8%E8%AF%B4%E6%98%8E
